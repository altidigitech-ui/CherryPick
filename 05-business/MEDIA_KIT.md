CherryPick is the masked AI persona that roasts startup culture and educates the developer community. With a Venetian mask, sharp wit, and no-BS attitude, CherryPick delivers brutal truths about SaaS building, indie hacking, and tech entrepreneurship.

**Tagline:** *"The AI that roasts your startup so the market doesn't have to."*

## Content formats disponibles

### Format 1 — Intégration dans un roast (8-15s)

Le produit/outil de la marque est intégré naturellement dans un roast CherryPick.
*Exemple : "You're using [competitor] when [brand product] exists? That's the real fail."*

| Détail | Info |
|--------|------|
| Durée | 8-15 secondes |
| Plateformes | TikTok + YouTube Shorts + Instagram Reels |
| Livrable | 1 vidéo adaptée aux 3 plateformes |
| Délai de production | 48-72h |

### Format 2 — Éducatif avec mention (1-2 min)

Contenu éducatif approfondi (Format E) avec une mention naturelle du produit/outil.
*Exemple : deep dive sur "les 3 outils que les top indie hackers utilisent" avec le produit en position #1.*

| Détail | Info |
|--------|------|
| Durée | 1-2 minutes |
| Plateformes | TikTok + YouTube Shorts + Instagram Reels |
| Livrable | 1 vidéo longue adaptée aux 3 plateformes |
| Délai de production | 72h-1 semaine |

### Format 3 — Série sponsorisée (3-5 vidéos)

Série de vidéos autour du produit/thème de la marque sur 1-2 semaines.

| Détail | Info |
|--------|------|
| Durée | Mix formats courts + longs |
| Plateformes | TikTok + YouTube Shorts + Instagram Reels |
| Livrable | 3-5 vidéos |
| Délai de production | 1-2 semaines |

## Marques idéales

CherryPick est pertinent pour les marques qui ciblent les développeurs et builders :

- **Outils SaaS** : Vercel, Railway, Supabase, Stripe, Resend, etc.
- **Outils dev** : GitHub Copilot, Cursor, JetBrains, Postman, etc.
- **Formation** : bootcamps, cours en ligne, plateformes d'apprentissage
- **Hardware** : laptops dev (Framework, Apple, Lenovo), périphériques
- **Hosting/Cloud** : AWS, GCP, DigitalOcean, Fly.io, etc.
- **VPN / Sécurité** : NordVPN, Surfshark, 1Password, etc.
